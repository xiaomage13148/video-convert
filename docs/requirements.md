# Video Convert Pro — 需求规划文档

> 将现有 Node.js CLI 视频批量转换工具，改造成 Vue 3 + Electron 桌面应用，提供精美的图形化操作界面。

---

## 一、项目背景

当前工具是一个纯命令行脚本，存在以下痛点：
- 目录路径硬编码，切换目录需改代码
- 无进度反馈，无法知道转换到哪了、还要多久
- 无参数配置界面，修改编码参数需直接改源码
- 转换失败无重试机制
- 无日志持久化，关闭终端后无法回溯

---

## 二、技术选型

| 层级 | 技术 | 理由 |
|------|------|------|
| 框架 | **Vue 3** + Composition API | 生态成熟、响应式系统适合实时进度更新 |
| 构建 | **Vite** | 极速冷启动、原生 ESM |
| UI 组件库 | **TDesign Vue Next** | 腾讯出品、风格简洁现代、组件丰富 |
| 桌面容器 | **Electron** | 需要文件系统访问 + 调用本地 ffmpeg |
| 状态管理 | **Pinia** | Vue 3 官方推荐、轻量 |
| 进度解析 | 解析 ffmpeg stderr | 实时提取转换进度、速度、剩余时间 |

## 三、功能需求

### 3.1 目录管理

| 功能 | 描述 |
|------|------|
| 选择输入目录 | 通过系统文件夹对话框选择视频所在目录 |
| 目录记忆 | 保存最近使用的 5 个目录路径，下次打开可快速选择 |
| 输出目录配置 | 默认与输入同目录，支持自定义输出目录 |
| 输出目录自动创建 | 若输出目录不存在，自动创建 |

### 3.2 文件扫描与展示

| 功能 | 描述 |
|------|------|
| 自动扫描 | 选择目录后自动扫描所有支持格式的视频文件 |
| 支持格式 | `.ts` `.mkv` `.avi` `.mov` `.wmv` `.flv` `.webm` `.m4v` `.mpg` `.mpeg` `.3gp` `.m2ts` `.mts` |
| 文件信息展示 | 文件名、格式、大小、编码格式（ffprobe 检测） |
| 文件筛选 | 按格式筛选、搜索文件名 |
| 勾选机制 | 默认全选，支持单选/多选/全选/反选 |
| 实时预检 | 转换前显示每个文件的预估输出路径，检测是否已有同名 mp4 存在（标记"将覆盖"） |

### 3.3 转换任务管理

| 功能 | 描述 |
|------|------|
| 批量启动 | 一键启动所有已勾选文件的转换 |
| 并发控制 | 可调节同时转换的任务数（1~6，默认3） |
| 任务状态 | 等待中 → 转换中 → 成功 / 失败 |
| 队列可视化 | 展示等待队列、活跃任务、已完成列表 |
| 单任务控制 | 支持单个任务取消 |
| 全局控制 | 全部暂停 / 全部继续 / 全部取消 |
| 失败重试 | 失败任务支持一键重试 |

### 3.4 实时进度展示

| 功能 | 描述 |
|------|------|
| 总体进度 | 顶部展示全局进度条（已完成/总数 + 百分比） |
| 单文件进度 | 每个任务卡片展示独立进度条 |
| 进度信息 | 当前百分比、转换速度、已用时间、预估剩余时间 |
| 颜色反馈 | 转换中=蓝色、成功=绿色、失败=红色、等待=灰色 |

### 3.5 编码参数配置

| 功能 | 描述 |
|------|------|
| 预设方案 | 提供「极速」「均衡」「高质量」三档预设 |
| 编码器选择 | h264_nvenc（默认）/ hevc_nvenc / libx264 / libx265 |
| 自定义参数 | 预设(preset)、质量因子(CQ/CRF)、码率上限、Profile、Level |
| 音频选项 | 直接复制（默认）/ 重新编码为 AAC |
| 硬件加速 | 开关 CUDA 硬件解码、手动选择解码器 |
| 参数持久化 | 保存用户自定义参数，下次打开自动加载 |

**预设方案详情：**

| 预设 | preset | CQ/CRF | 码率 | 说明 |
|------|--------|--------|------|------|
| 极速 | p1 | 28 | 不限 | 最快速度，文件略大 |
| 均衡 | p4 | 23 | 不限 | 速度与质量平衡（默认） |
| 高质量 | p7 | 18 | 不限 | 最高质量，速度较慢 |

### 3.6 后处理选项

| 功能 | 描述 |
|------|------|
| 删除原文件 | 默认开启，转换成功后自动删除原始文件 |
| 覆盖确认 | 若输出路径已有同名文件，弹窗确认是否覆盖 |
| 完成通知 | 全部转换完成后系统通知提醒 |

### 3.7 日志面板

| 功能 | 描述 |
|------|------|
| 实时日志 | 展示每个任务的 ffmpeg 输出信息 |
| 日志级别 | INFO / WARN / ERROR 三级，可按级别筛选 |
| 日志搜索 | 支持关键词搜索 |
| 导出日志 | 一键导出为 .txt 文件 |
| 自动滚动 | 日志自动滚动到底部，可手动暂停滚动 |

### 4.2 页面说明

#### 转换页（首页）
- **顶部**：目录选择区域，显示当前路径、最近使用列表
- **进度区**：全局进度条 + 统计信息（成功/失败/等待数）
- **文件列表**：卡片式布局，每个文件一张卡片，展示文件信息和实时进度
- **底部操作栏**：开始、暂停、取消、重试等操作按钮

#### 设置页
- **预设方案**：一键切换三档预设
- **编码参数**：编码器、preset、质量、码率等详细配置
- **硬件加速**：CUDA 开关、解码器选择
- **后处理**：删除原文件开关、覆盖确认开关
- **并发数**：滑块调节 1~6

#### 日志页
- **筛选栏**：按级别筛选 + 搜索框
- **日志列表**：虚拟滚动的高性能日志列表
- **操作**：导出、清空、暂停滚动

### 4.3 设计风格

- **色调**：深色主题为主（护眼、视频工作者偏好），支持浅色切换
- **配色**：主色采用科技蓝 (#0052D9)，辅以绿色成功、红色失败
- **动效**：进度条平滑过渡、卡片进出动画、按钮微交互
- **布局**：左侧固定导航（可折叠） + 右侧内容区自适应

### IPC 通信协议

| 通道 | 方向 | 描述 |
|------|------|------|
| `scan:directory` | 渲染→主 | 请求扫描目录 |
| `scan:result` | 主→渲染 | 返回扫描结果 |
| `convert:start` | 渲染→主 | 启动转换任务 |
| `convert:pause` | 渲染→主 | 暂停任务 |
| `convert:cancel` | 渲染→主 | 取消任务 |
| `convert:retry` | 渲染→主 | 重试失败任务 |
| `convert:progress` | 主→渲染 | 实时进度推送 |
| `convert:status` | 主→渲染 | 任务状态变更 |
| `convert:complete` | 主→渲染 | 全部完成通知 |
| `settings:load` | 渲染→主 | 加载设置 |
| `settings:save` | 渲染→主 | 保存设置 |
| `dialog:directory` | 渲染→主 | 打开目录选择对话框 |
| `gpu:info` | 渲染→主 | 获取 GPU 信息 |

## 九、风险与对策

| 风险 | 对策 |
|------|------|
| ffmpeg 进度解析不准 | 解析 stderr 的 `time=` 字段，结合文件总时长计算百分比 |
| CUDA 不可用时崩溃 | 首次启动检测 GPU，不支持时自动回退到 CPU 编码 |
| 大量文件时列表卡顿 | 使用虚拟滚动（TDesign 虚拟列表或 vue-virtual-scroller） |
| Electron 打包体积大 | 使用 asar 压缩 + 按需引入 TDesign 组件 |
